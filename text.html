Text

Content is king!

Content is important to any web site and web application. Making sure that it has the proper markup should have high priority.
Text should be in paragraphs <p>, headings in <h1><h2> etc. Giving thought to which type of list you need to use. Making sure your tabular data is setup in a table. The most important is that you mark up your consistently. So we've set up some examples to get you started.
