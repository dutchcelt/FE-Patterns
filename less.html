Less

Maintainable CSS!

One of the big problems CSS has is that it isn’t very maintainable. It can very easily get away from you. LESS allows you to define variables for any property value or even the entire contents of an existing rule. Creating a custom skin for a web site has never been this easy.
The most important units are actually @view and @font-scale. You may also notice that all vertical units are relative and horizontal units are absolute (pixels). This is due to the web’s nature of content being vertical and constrained in the width. Using media queries will allow you to create semi fluid layout.

Note: The pattern may evolve into a fully a fluid layout.

Layout

// Base units that set overall proportions ------------------------------------ @view: 1200; // VIEWPORT target based on pixels @w: @view / 100px; // Smallest horizontal unit @h: 0.2em; // Smallest vertical unit @em: @h*7; // Vertical space for boundries like top & bottom padding @block: @w * 2; // White space for paddings or margins @text-block: @view * 48%; // alt: 52em; @max-width: @view * 1px; // alt: @w * 80 = 960 @l: 1px; // Line or border @rad: @w/3; // Standard radius // Form elements ------------------------------------------------------------- @form-width: @text-block; @field-width: @form-width; @input-padding-height: @em/4; @input-padding-width: @block/6; @text-input-padding: @input-padding-height @input-padding-width; @label-width: 100%; @text-input-width: 100%; @label-orientation: none; /* float value: left, right, none */ @label-text-orientation:left; // Box elements -------------------------------------------------------------- .box { margin: 0 0 @interline ; padding: @interline @block; } // Use this to have block elements override the '.panel' padding. // This is great for headings, <header> and <footer> tags. .box-inset { margin: 0 - @interline 0 - @block; padding: @interline/2 @block; }

Typography

// Font elements ------------------------------------------------------------- @font-root: "../fonts/"; @font-face { font-family: 'fontname'; src: url('@{font-root}fontname.woff') format('woff'); font-weight: normal; font-style: normal; } @body-scale: 100%; // Used on the body element @transport: 1.5; // For line height (no units!) @interline: @h*5; // Interline / leading http://en.wikipedia.org/wiki/Interline_spacing @typeface: "fontname", Arial, sans-serif; @typeface-header: "fontname", Arial, sans-serif; @em-scale: 0.9em; @header-weight: normal; @header-lineheight: 1.4; /* TYPOGRAPHIC FONT SCALE (DO NOT EDIT) ----------------------------------------- */ .nonpareil { font-size: @em-scale * 0.6; } .minion { font-size: @em-scale * 0.7; } .brevier { font-size: @em-scale * 0.8; } .galliard { font-size: @em-scale * 0.9; } .garamond { font-size: @em-scale * 1; } .small-pica { font-size: @em-scale * 1.1; } .pica { font-size: @em-scale * 1.2; } .augustin { font-size: @em-scale * 1.4; } .great-primer { font-size: @em-scale * 1.8; } .double-pica { font-size: @em-scale * 2.1; } .two-line-pica { font-size: @em-scale * 2.4; } .double-great-primer { font-size: @em-scale * 3.6; }
