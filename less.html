LESS

Maintainable CSS!

One of the big problems CSS has is that it isn’t very maintainable. It can very easily get away from you. LESS allows you define variable for any property value or even the entire contents of an existing rule. Creating a custom skin for a web site has never been this easy.
The most important size unit are actually the @grid-unit and the @font-scale. You may also notice that all vertical units are relative and horizontal units are absolute (pixels). This is due to the web’s nature of content being vertical and constrained in the width.

Grid units

// Base units that set overal proportions ------------------------------------ @grid-unit: 12px; @unit: 0.2em; @em: @unit*7; @base-height: @em; @base-width: @grid-unit * 4; @block-space: @base-width / 2; @text-block-width: @base-width * 11; @max-width: 960px; @line: 1px; @top: 0 - @line; @bottom: @line; @rad: @grid-unit / 3; // border radius setting
