LESS

Maintainable CSS!

One of the big problems CSS has is that it isn’t very maintainable. It can very easily get away from you. LESS allows you define variable for any property value or even the entire contents of an existing rule. Creating a custom skin for a web site has never been this easy.
The most important units are actually @view and @font-scale. You may also notice that all vertical units are relative and horizontal units are absolute (pixels). This is due to the web’s nature of content being vertical and constrained in the width. Using media queries will allow you to create semi fluid layout.

Grid units

// Base units that set overal proportions ------------------------------------ @view: 1200; // VIEWPORT target based on pixels @w: @view / 100px; // Smallest horisontal unit @h: 0.2em; // Smallest vertical unit @em: @h*7; // Line transport @block: @w * 2; // White space for paddings etc @text-block: @view * 48%; // alt: 52em; @max-width: @view * 1px; // alt: @w * 80 = 960 @l: 1px; // Line @rad: @w/3; // Standard radius

Typography

// Font elements ------------------------------------------------------------- @font-root: "../fonts/"; @font-face { ... } @font-face { ... } @font-scale: 82.5%; @transport: 1.6; @line: @h*5; @typeface: Arial, sans-serif; @typeface-header: "LobsterTwoBoldItalic", Georgia, serif; @typeface-brand: "OstrichBlack", Arial, sans-serif;
