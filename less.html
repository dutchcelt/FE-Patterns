LESS

Maintainable CSS!

One of the big problems CSS has is that it isn’t very maintainable. It can very easily get away from you. LESS allows you to define variables for any property value or even the entire contents of an existing rule. Creating a custom skin for a web site has never been this easy.
The most important units are actually @view and @font-scale. You may also notice that all vertical units are relative and horizontal units are absolute (pixels). This is due to the web’s nature of content being vertical and constrained in the width. Using media queries will allow you to create semi fluid layout.

Note: The pattern may evolve into a fully a fluid layout.

Grid units

// Base units that set overal proportions ------------------------------------ @view: 1200; // VIEWPORT target based on pixels @w: @view / 100px; // Smallest horisontal unit @h: 0.2em; // Smallest vertical unit @em: @h*7; // Line transport @block: @w * 2; // White space for paddings etc @text-block: @view * 48%; // alt: 52em; @max-width: @view * 1px; // alt: @w * 80 = 960 @l: 1px; // Line @rad: @w/3; // Standard radius

Typography

// Font elements ------------------------------------------------------------- @font-root: "../fonts/"; @font-face { ... } @font-face { ... } @font-scale: 82.5%; @transport: 1.6; @line: @h*5; @typeface: Arial, sans-serif; @typeface-header: "LobsterTwoBoldItalic", Georgia, serif; @typeface-brand: "OstrichBlack", Arial, sans-serif;

Forms

// Form elements ------------------------------------------------------------- @form-width: @text-block; @field-width: @text-block; @input-padding-height: @em/4; @input-padding-width: @block/6; @text-input-padding: @input-padding-height @input-padding-width; @label-width: 100%; @text-input-width: 100%; @label-orientation: none; /* float value: left, right, none */ @label-text-orientation:left;

Boxes & panels

// Box elements -------------------------------------------------------------- .box { margin: 0 0 @line ; padding: @line @block; } // Use this to have block elements override the '.panel' padding. // This is great for headings, <header> and <footer> tags. .box-inset { margin: 0 - @line 0 - @block; padding: @line/2 @block; }

Icons

// ICONS --------------------------------------------------------------------- // Icons for :before or :after & set parent element to position relative @icon-root: "../images/icons/"; // Arguments: // 1. absolute position top (default: 0px) // 2. absolute position left (default: 0px) // 3. filename to icon(s) (default: jQUERY UI's icons) // 4. Icon size (default: 16px) .icon(@postop:0, @posleft:0, @file:'ui-icons_454545_256x240.png', @icon-size:16px) { position: absolute; top: @postop; left: @posleft; content: ""; padding: @icon-size/2; background-image: url('@{icon-root}@{file}'); background-repeat: no-repeat; }
